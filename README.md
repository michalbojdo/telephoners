# SKN Telephoners<br> strona internetowa - frontend 

W ```vite.config.ts``` w ```defineConfig```

ustawić port aplikacji

``` 
server:{
    allowedHosts:true,
    port: <port aplikacji>
  }
  ```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

```sh
npm run preview
```
